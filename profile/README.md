# Conway USP

Seja bem-vindo, nós somos a **Conway**, uma liga acadêmica nascida na Escola de Artes, Ciências e Humanidades da Universidade de São Paulo. Somos compostos
principalmente por estudantes de Sistemas de Informação, mas recebemos alunos de diversos cursos e até de diferentes institutos. Nossa missão é democratizar o conhecimento relacionado
a área de **desenvolvimento de jogos** e **computação gráfica**.

Em nosso Github você encontrará nossos principais projetos, incluindo cursos, trilhas, jogos e outros materiais.

Veja abaixo para nossos principais repositórios:

## Trilhas

Nossos materiais de estudos para capacitação de nossos membros,

### Desenvolvimento de Jogos

- [Hiten Hagomoro](https://github.com/ConwayUSP/Hiten_Hagoromo), introdução a linguagem Lua e ao framework LÖVE 2D.
- [Introdução ao Desenvolvimento de Jogos 2D](https://github.com/ConwayUSP/Intro_Games2D) com o framework LÖVE 2D.

- [Introdução ao Desenvolvimento de Jogos 3D](https://github.com/ConwayUSP/Intro_Games3D) com o Godot Game Engine.
- [Lucky Crab](https://github.com/ConwayUSP/Lucky_Crab), curso introdutório a linguagem Rust, ao Godot Game Engine e a _binding_ Godot-Rust.

- [Sea of C](https://github.com/ConwayUSP/Sea-of-C), curso de introdução ao framework Raylib com a linguagem C.

### Computação Gráfica

- [Estado da Arte](https://github.com/ConwayUSP/Estado-da-Arte), curso de introdução à OpenGL e conceitos de computação gráfica.
